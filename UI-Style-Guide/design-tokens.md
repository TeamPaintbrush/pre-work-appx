# Design Tokens

Centralize colors, fonts, spacing, and other design values for consistency.

Example:
```js
export const tokens = {
  colors: {
    primary: '#2563eb',
    secondary: '#64748b',
    accent: '#f59e42',
    error: '#ef4444',
    success: '#22c55e',
  },
  fontFamily: 'Inter, sans-serif',
  spacing: {
    sm: '8px',
    md: '16px',
    lg: '32px',
  },
};
```
